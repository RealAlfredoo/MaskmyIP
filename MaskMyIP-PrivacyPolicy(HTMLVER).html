01

Introduction

This Privacy Policy describes how MaskMyIP ("the Extension"), developed by Oskar Pękala (aka. "RealAlfredoo") · github.com/RealAlfredoo/MaskmyIP, handles information when you use the extension in your browser.

By installing and using MaskMyIP, you agree to the terms described in this policy. If you do not agree, please uninstall the extension.

02

Data We Do NOT Collect

MaskMyIP is built with a strict no-data-collection philosophy. We do not collect, store, transmit or share:

Your IP address · browsing history · websites you visit · personal information of any kind · usage statistics · device information · cookies or tracking identifiers.

The extension operates entirely within your local browser environment. No data ever leaves your device to our servers — because we don't have any servers.

03

Local Storage

MaskMyIP stores your preferences (selected mode, language, toggle state) using your browser's built-in local storage. This data:

— stays exclusively on your device
— is never transmitted anywhere
— can be cleared at any time by removing the extension

What is stored locally:
Selected protection mode (Pattern / Auto / Manual) · UI language preference (EN/PL) · Protection on/off state · Masked IP count (statistics, local only)
04

Third-Party API — ipify.org

When you use Auto-Detect Mode, the extension makes a single request to ipify.org — a public, free IP lookup API — to retrieve your current public IP address.

This request is made directly from your browser to ipify.org. MaskMyIP does not intercept, store or process this data — the IP is used only locally to determine what to mask on webpages.

API used: api.ipify.org (primary) · api.my-ip.io (fallback)
Data sent: Standard browser request (no additional data)
Data received: Your public IP address (used locally, never stored by us)
ipify.org privacy policy: ipify.org

If you do not wish to use this API, simply use Pattern Mode or Manual Mode instead — neither makes any external requests.

05

Permissions

MaskMyIP requests the following browser permissions and here is exactly why:

storage — to save your preferences locally in the browser
activeTab — to scan and mask IP addresses on the current page
scripting — to inject the masking script into webpages
<all_urls> — to work on all websites you visit

None of these permissions are used to collect, transmit or analyze your data. They exist solely to enable the core functionality of IP masking.

06

Children's Privacy

MaskMyIP does not knowingly collect any information from anyone, including children under the age of 13. Since we collect no data whatsoever, this extension is safe for all age groups.
