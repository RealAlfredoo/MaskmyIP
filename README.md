
### 🛡️ MaskMyIP — Hide Your IP on Any Webpage

MaskMyIP automatically detects and masks your IP address across the web, replacing it with a clean **MaskMyIP** badge wherever it appears. Lightweight, fast, and privacy-first.

---

### Three Masking Modes

**⚡ Auto Mode**
MaskMyIP fetches your real public IP address using the [ipify.org](https://api.ipify.org) API — a trusted, open, and privacy-respecting service. Once detected, any occurrence of your IP on any webpage is instantly replaced. No configuration needed.

**🔍 Pattern Mode** *(default),(best)*
Don't want to rely on an external API? Pattern mode scans every page for any text matching the format of an IP address (e.g. `192.168.0.1`) and masks them all — regardless of whether it's your IP or not. Maximum coverage, zero API calls.

**✏️ Manual Mode**
Know your IP already? Enter it yourself and MaskMyIP will hide exactly that address across all pages. Full control, no lookups.

### How It Works

Whenever your IP is detected on a page, MaskMyIP replaces it with a subtle green **MaskMyIP** badge inline — preserving the page layout while keeping your address hidden. Works on dynamically loaded content too (React, AJAX, SPAs).

---

### Features at a Glance

- Runs automatically on every page you visit
- Masked IP in the extension popup (click to reveal)
- Live counter: hidden IPs & protected pages
- 🇺🇸 English / 🇵🇱 Polish interface
- Works on Chrome, Edge, Firefox, and Brave

### ⚠️ Disclaimer

MaskMyIP masks IP addresses visible in webpage content. It is not a VPN and does not hide your IP at the network level. Due to technical limitations, some dynamically generated or canvas-rendered content may not be masked. We are not liable for any IP address exposure that may occur. Use at your own risk.
